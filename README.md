# hcot 

hcot stands for 'HTMX + Django-cotton' and is a boilerplate project template made by Henry Sheffield, it comes with an authentication form, user profile management, and a few example HTMX components to get you started. It also includes Django-cotton for easy admin customization.

## Daisy UI installation (for macs)
cd hcot/static/css && curl -sL daisyui.com/fast | bash

hcot/static/css/tailwindcss -i hcot/static/css/input.css -o hcot/static/css/output.css --watch
