{% comment %} Toast Notifications Container This component displays Django messages as toast notifications in the bottom-right corner. Automatically converts Django message levels to toast types. Usage in base.html: Django message tags mapping: - success -> success toast - error/danger -> error toast - warning -> warning toast - info/debug -> info toast {% endcomment %}
{% if messages %} {% for message in messages %} {% with message_type=message.tags|default:"info" %} {% if "success" in message_type %} {% elif "error" in message_type or "danger" in message_type %} {% elif "warning" in message_type %} {% else %} {% endif %} {% endwith %} {% endfor %} {% endif %}
